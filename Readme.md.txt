#Generador de Contraseñas
Proyecto con tecnologias como HTML,Javascript y CSS
¿Porque?:Desarrolle esta herramienta porque en ciberseguridad se suele hablar que necesitamos cierta  robustez a la hora de crear una contraseña de forma segura o fortalecer nuestro entorno virtual, con el fin de entender la generacion de datos de forma aleatoria y la importancia en la seguridad de ellos,el objetivo principal es mitigar riesgos de ataques de fuerza bruta y externos a usuarios,permitiendo al usuario alternar diversas contraseñas para asegurar sus cuentas e archivos digitales.
Para asegurar la proteccion de archivos digitales y mejorar la tipologia de nuestras cuentas de usuario de una forma mas segura.
¿Como?:Utilize HTML y CSS Con asistente para enfocarme en la eficiencia del codigo
se utilizo la logica de la programacion en hechos como la implementacion de algoritmo en Javascript para la seleccion aleatoria de diversos caracteres.
Interfaz de usuario: amigable con diseño limpio y funcional mediante HTML Y CSS.
Optimizacion:Utilice asistentes de inteligencia artificial para estructurar las ideas,logica y mejor eficencia en el codigo y seguir mejores practicas en cuanto a estructuras.
Carasteristicas:Generacion de cadenas alfanumericas,inclusion opcional de caracteres especiales.
