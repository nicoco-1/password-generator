# 🔐 Generador de Contraseñas Seguras

Un generador de contraseñas seguras interactivo y moderno construido con **HTML, CSS y JavaScript**.

## ✨ Características

- ✅ **Generación de contraseñas aleatoriamente seguras**
- ✅ **Longitud personalizable** (4-32 caracteres)
- ✅ **Opciones configurables:**
  - Mayúsculas (A-Z)
  - Minúsculas (a-z)
  - Números (0-9)
  - Símbolos especiales (!@#$%^&*)
- ✅ **Indicador de fortaleza** en tiempo real (Débil, Media, Buena, Muy Fuerte)
- ✅ **Copiar al portapapeles** con un clic
- ✅ **Interfaz responsive** (funciona en mobile, tablet y desktop)
- ✅ **Diseño moderno** con gradientes y animaciones

## 🚀 Uso

1. **Abre** el archivo `index.html` en tu navegador
2. **Selecciona** qué tipos de caracteres deseas incluir
3. **Ajusta** la longitud de la contraseña con el slider
4. **Haz clic** en "Generar Contraseña"
5. **Copia** la contraseña generada al portapapeles

## 🎨 Características Técnicas

- **HTML5 Semántico**: Estructura accesible y bien organizada
- **CSS3 Avanzado**: Gradientes, animaciones y flexbox
- **JavaScript Vanilla**: Sin dependencias externas
- **Async/Await**: Para copiar al portapapeles de forma segura
- **Responsive Design**: Media queries para todos los tamaños

## 🔒 Seguridad

Las contraseñas se generan localmente en tu navegador. **Ninguna contraseña se envía a servidores externos**. Todo funciona 100% en el cliente.

## 🛠️ Requisitos
